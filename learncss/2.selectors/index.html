Selectors in html

Element Selector

The element selector selects HTML elements based on the element name

Id Selector

The id selector is used id attribute of an HTML element to select a specific element. The id of an element is unique within the page ,so the id selector is used to select one unique element!

Class Selector

The class selector select the HTML elements with specific class attribute. To select element with specific class, write a period(.)charactor followed by the class name.
